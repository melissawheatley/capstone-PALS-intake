If you want an element to have a light grey background, give it a .greyBG class.
These colored background classes have a base padding of 1rem.

If you want an element to have a steel grey background, give it a .steelBG class.
You will need to set the text color to white.

If you want an element to have a light blue background, give it a .blueBG class.
